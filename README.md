### 3. **Forest Fire Detection System** - IoT & Sensor Networks
**[Live Deployment](https://forestguard.rishabhj.in/) | Status: In Progress**

A cost-effective **Wireless Sensor Network (WSN)** system designed for autonomous forest fire detection in remote areas.

**Key Features:**
- 🔬 Dual-sensor verification: MQ-2 gas sensor + DHT22 temperature/humidity
- 🧠 STM32F303RET6 microcontroller with 32-bit ARM Cortex-M4
- 📡 Zigbee mesh networking (IEEE 802.15.4) for reliable communication
- 🔋 Ultra-low power firmware for remote deployment
- 🔔 Dual alerts: Local buzzers/LEDs + remote 20x4 LCD display
- ✅ Minimized false alarms with intelligent threshold logic

**Impact:** Zigbee outperforms Wi-Fi in power efficiency and range, making it ideal for infrastructure-poor forest environments.

**Tech Stack:** STM32 • C • Zigbee • IoT • Embedded Systems
