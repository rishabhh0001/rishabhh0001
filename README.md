### 2. **Music Visualizer & Equalizer** - @rishabh0001/visualizer
**[Live Deployment](https://music.rishabhj.in/) | [GitHub](https://github.com/rishabhh0001/visualizer)**

An interactive music visualizer and equalizer application that transforms audio into stunning real-time visual experiences.

**Key Features:**
- 🎵 Real-time audio visualization with dynamic frequency analysis
- 🎚️ Interactive 10-band equalizer with custom presets
- 🎨 Multiple visualization modes and color schemes
- 📊 Frequency spectrum analyzer with smooth animations
- 🎧 Support for multiple audio sources
- ⚡ High-performance WebGL rendering

**Tech Stack:** TypeScript • React • Web Audio API • WebGL
